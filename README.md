# pandaCoin
一款基于海豚BaaS平台快速开发的一套区块链系统。

## 项目简介

------

    panda一款基于海豚BaaS平台快速开发的一套区块链系统。基于平台更简单的区块链数据服务。以更少的代价更低的学习成本，快速实现真正的区块链应用。这里以熊猫币为例子实现数字货币系统的大部分功能。

其功能主要实现：
> * 创建主币种panda币
> * 创建子币种
> * 主币种转账（panda币转账消耗panda币作为gas）
> * 子币种转账（子币种转账消耗panda币作为gas）
> * ECDSA钱包
> * 转账客户端类
> * 转账接口
> * 企业区块更新
> * 地址余额查询
